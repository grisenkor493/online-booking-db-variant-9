# Проектирование базы данных для языкового центра

Студент: Руслан, группа ИСП-21.
Вариант: 9.

Предметной областью является автоматизация процессов учета, тестирования и записи клиентов в языковой центр. База данных обрабатывает входное тестирование, распределение по уровням и контролирует заполненность групп, формируя очередь при отсутствии мест.

Стек технологий: СУБД MySQL, SQL.

Структура репозитория включает несколько основных разделов. В корне находится файл описания и файл исключений для системы контроля версий. Папка sql содержит основной скрипт развертывания таблиц базы данных. В папке queries располагаются файлы с отдельными аналитическими запросами. Раздел docs отведен под текстовые отчеты в форматах PDF и DOCX, а внутри него есть подпапка images для хранения скриншотов результатов выполнения запросов. Графическая ER-диаграмма вынесена в отдельную директорию diagrams.

Инструкция по развёртыванию базы данных:
1. Склонируйте репозиторий на локальный компьютер.
2. Откройте MySQL Workbench или другую среду управления БД.
3. Выполните скрипт из файла sql/schema.sql для создания структуры таблиц и заполнения их тестовыми данными.

Отчет о проделанной работе находится в папке docs.
